Research - Project List


Design and Development of Vision based Compact Automated Guided Vehicle
  • August,2017 - present
  • Group project guided by Dr Suril Shah, Dr Kaushal Desai
  • autonomous navigation, computer vision

This project is part of my B.Tech Thesis for pre-final year.The aim of this project is to design and develop a compact Automated Guided Vehicle with industry standards to carry payload of 700kg which is fully autonomous. I am involved in developing state of art SLAM algorithm with low cost vision system and developing path planning algorithm for the vehicle.

Read More
Investigation of multi-directional 3D printer for printing goods with improved quality.
  • February 2017 - May 2017
  • Group project guided by Dr Suril Shah, Dr Kaushal Desai
  • path planning, motor control

This project is part of my B.Tech Thesis for sophomore year.The aim of the project was to develop a multi-directional 3D printer through Stewart Gough Platform so the printed parts have improved strength and also take less time to build further the parts can be directly printed to curved surfaces. I was involved in controlling the extruder at given rpm so that it prints parts smoothly and designing a cavity such that it minimizes the heat loss.

Read More
Mechanism to allow paraplegic person to stand up in his wheelchair in order to play golf.
  • February 2017 - April 2017
  • Group project
  • Kinematic analysis, Motion Capture using VICON

This project was part of course project of the course Kinematics of Machines and Mechanisms. The aim of the project was to develop a wheelchair constituting a simple linkage such that the above task is accomplished and to do Kinematic Analysis of the same linkage. I was involved in capturing motion of Human while standing up using VICON motion capture system and simulating it in MATLAB to get initial trajectory points to design linkage.

Read More
Escalade, IIT Guwhati
  • May 2016 - July 2016
  • Group project guided by Dr B.Ravindra
  • Designing, motor control

The aim of this project was to design and develop a robot that can pick up boxes weighing approx 200kg and lift the boxes up to certain height between two upright frames. This project was part of National Robotics Competition Escalade which is conducted by IIT Guwhati. I was involved in designing the gripper mechanism and controlling the robot. We were among top three teams in Jaipur Prelims round.
